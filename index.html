Courier Ops — Weather Control

One-click weather settings automation for Bolt Admin Panel.
No extensions, no servers — works directly in your browser via bookmarklet.

1 Set up the Bookmarklet (one-time, 30 sec)

  1. 1Show the Bookmarks Bar in Chrome: Cmd+Shift+B (Mac) or Ctrl+Shift+B (Windows)
  2. 2Drag the blue button below onto your Bookmarks Bar:

Drag this button to your Bookmarks Bar:

COps Weather Dashboard
Done! The bookmarklet is now saved in your browser. No installation, no extension, no admin rights needed. Works even with managed Chrome.

2 How to use

  1. 1Open admin-panel.bolt.eu and make sure you're logged in
  2. 2Click "COps Weather Dashboard" in your Bookmarks Bar
  3. 3A full dashboard overlay appears with all 37 cities
  4. 4Select your name, then click Good / Bad / Harsh on any city
  5. 5The bookmarklet opens the city settings page, merges JSON, and clicks Update — automatically!

How it works under the hood

  • Bookmarklet injects a dashboard overlay on top of the admin panel page
  • When you click Apply, it opens the city settings in a popup window (same domain = full access)
  • Fetches target settings from the GitHub repo, deep-merges with current settings
  • Writes merged JSON to the editor and clicks Update — same as the local Selenium scripts
  • Your browser handles the SSL certificate and authentication automatically
  • Status is tracked locally and shared via the GitHub repo
